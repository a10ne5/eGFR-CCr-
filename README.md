# eGFR・CCR・BSA計算アプリ

eGFR・CCR・BSAを計算する、シンプルな静的ウェブアプリです。  
難しい準備なしで、まずはブラウザで開いて使える形にしてあります。

- eGFR: 年齢・性別・血清クレアチニンで計算
- CCR: 年齢・性別・血清クレアチニン・体重で計算
- BSA: 身長・体重から藤本式で計算

## 使い方

### いちばんおすすめの使い方

オフラインで使いたいなら、`GitHub Pages` などで一度だけSafariから開いて、ホーム画面に追加して使う方法がいちばん楽です。

この方法では、

- 最初の1回だけネット接続が必要
- その後はオフラインでも使いやすい
- iPhoneで普通のアプリのように開きやすい

というメリットがあります。

### オフライン対応版の使い方

1. このアプリを `https://` で始まるURLで開く
2. Safariで [index.html](/Users/wadakou/Desktop/Codex/eGFRWebApp/index.html) 相当のページを表示する
3. 画面上の「初回だけSafariでオンライン表示すると...」の表示を確認する
4. Safariの共有メニューから「ホーム画面に追加」を押す
5. 以後はホーム画面のアイコンから開く

注意:

- オフライン保存の仕組みは `ファイルを直接開く方法` では動きません
- `https://` で始まるページとしてSafariで開く必要があります

### 1. パソコンで開く方法

いちばん簡単なのは、`index.html` をブラウザで開く方法です。

1. Finderでこのフォルダを開く
2. `index.html` をダブルクリックする
3. ブラウザでアプリが開く

ファイル:

- `index.html`
- `styles.css`
- `script.js`

## もう少しちゃんと見る方法

ターミナルでこのフォルダに移動して、次のコマンドを実行します。

```bash
python3 -m http.server 8000
```

そのあと、ブラウザで次を開きます。

```text
http://localhost:8000
```

## スマホで見る方法

方法はいくつかあります。初心者の方には次の2つがわかりやすいです。

### 方法1. GitHub Pagesで公開する

これは「ネット上に自分のアプリを置く方法」です。

1. GitHubにこのフォルダをアップする
2. GitHub Pagesを有効にする
3. 発行されたURLをスマホで開く

この方法だと、iPhoneでもAndroidでもすぐ確認できます。

### 方法2. パソコンとスマホを同じWi-Fiにつなぐ

1. パソコンで次のコマンドを実行する

```bash
python3 -m http.server 8000
```

2. パソコンのIPアドレスを確認する
3. スマホのブラウザで `http://パソコンのIPアドレス:8000` を開く

例:

```text
http://192.168.1.20:8000
```

## 計算式

### eGFR

日本人向けの血清クレアチニンベース式を使用します。

- 男性: `194 × Cr^-1.094 × 年齢^-0.287`
- 女性: `194 × Cr^-1.094 × 年齢^-0.287 × 0.739`

### CCR

Cockcroft-Gault式を使用します。

- 男性: `((140 - 年齢) × 体重) / (72 × 血清Cr)`
- 女性: `((140 - 年齢) × 体重) / (72 × 血清Cr) × 0.85`

### BSA

藤本式を使用します。

- `0.008883 × 身長(cm)^0.663 × 体重(kg)^0.444`

## 注意

- このアプリは参考計算用です
- 診断や治療判断を目的としたものではありません
- CCRのCockcroft-Gault式には限界があり、臨床用途では注意が必要です
- BSAの藤本式は身長をcm、体重をkgで入力する前提です
